## 🔧 IF PORT ISSUE HAPPENS AGAIN

Use the cleanup script:
```bash
kill-port-3001.bat
```

Or manually:
```bash
# Find process
netstat -ano | findstr :3001

# Kill it (replace PID with actual number)
powershell -Command "Stop-Process -Id PID -Force"
```
